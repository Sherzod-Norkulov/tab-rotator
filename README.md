# Tab Rotator

Гибкая ротация вкладок: общий интервал, индивидуальные таймеры, автообновление, профили, отдельное окно и импорт/экспорт конфигураций.

## Возможности
- Переключение вкладок по общему интервалу или по индивидуальным таймерам для каждой записи.
- Кастомный список URL с автооткрытием и опциональным отдельным окном мониторинга.
- Автообновление выбранных вкладок, исключения по доменам, последовательный или случайный порядок.
- Профили (сохранение/импорт/экспорт JSON).
- Бейдж на иконке с обратным отсчётом (можно отключить).

## Установка из исходников
1) Откройте `chrome://extensions` и включите «Режим разработчика».  
2) Нажмите «Загрузить распакованное расширение» и выберите каталог проекта.

## Права и данные
- `tabs` — переключение/открытие вкладок в выбранном окне.
- `storage` — локальное хранение настроек и профилей.
Данные не отправляются на сторонние серверы. Подробности: `PRIVACY_POLICY.md`.
