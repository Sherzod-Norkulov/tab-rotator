# Tab Rotator — Политика конфиденциальности

Обновлено: 2025-12-11

## Что делает расширение
- Переключает вкладки по таймеру.
- По желанию открывает заданный список URL в отдельном окне.
- Сохраняет настройки (интервалы, список URL, профили) локально в браузере.

## Какие данные мы собираем
- **Ничего не отправляем на сторонние серверы.**
- Локально (в `chrome.storage.local`) сохраняются только ваши настройки: интервалы, список ссылок, профили, флаги автообновления, порядок переключения и состояние автозапуска.
- Расширение запрашивает доступ `tabs` и `storage`, чтобы читать список открытых вкладок для переключения и сохранять настройки.

## Обработка и хранение
- Все данные хранятся локально в браузере пользователя.
- Данные не передаются третьим лицам и не используются для идентификации пользователя.
