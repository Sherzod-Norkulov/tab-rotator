# Tab Rotator — материалы для публикации в Chrome Web Store

## Коротко (до 132 символов)
Автопереключение вкладок с гибкими таймерами, профилями, отдельным окном мониторинга и автообновлением выбранных страниц.

## Полное описание (пример)
- Настраиваемый интервал переключения + индивидуальные таймеры для каждой страницы.
- Профили (Work/Monitoring/Reports) с быстрым переключением и импортом/экспортом JSON.
- Отдельное окно мониторинга, чтобы не трогать рабочие вкладки.
- Автообновление выбранных страниц, исключения по доменам, случайный или последовательный порядок.
- Бейдж на иконке показывает обратный отсчёт (можно отключить).
- Ничего не отправляет наружу, настройки хранятся локально (см. PRIVACY_POLICY.md).

## Категория
Продуктивность.

## Языки
ru (UI), en планируется.

## Разрешения из манифеста
- `tabs` — переключение и открытие вкладок в окне мониторинга.
- `storage` — сохранение настроек и профилей локально.

## Политика конфиденциальности
Файл: `PRIVACY_POLICY.md` (опубликуйте как публичную ссылку на GitHub или собственный хост и укажите URL в форме).

## Скриншоты (нужно подготовить)
- 3–5 штук 1280×800 или 1280×720 с разными состояниями (основной экран, профили, импорт/экспорт, отдельное окно).
- Минимум один должен быть 1280×800.
- Пример демо: assets/screenshots/demo.png (склеены ключевые блоки вместо s1–s4).

## Иконки (требование CWS)
- 128×128 обязательно, желательно 48×48 и 16×16. Поместить в `assets/icons/` и указать в манифесте.

## Архив для загрузки
1. Убедитесь, что в корне есть `manifest.json`, `background.js`, `popup.html`, `popup.js`, иконки и политика.
2. Соберите ZIP всего каталога расширения без лишних файлов:
   ```
   zip -r tab-rotator.zip manifest.json background.js popup.html popup.js PRIVACY_POLICY.md STORE_LISTING.md assets
   ```
3. Загрузите ZIP в Dev Console, заполните форму (описание, скриншоты, политика).

## Чек-лист перед публикацией
- [ ] Иконка 128×128 прописана в `manifest.json` (`"icons"`).
- [ ] Скриншоты 1280×800+ загружены.
- [ ] Политика опубликована по URL.
- [ ] Короткое и полное описание вставлены.
- [ ] Проверить работу горячих клавиш, профилей, исключений и бейджа в прод-сборке.
