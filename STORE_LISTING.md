# Tab Rotator — материалы для публикации в Chrome Web Store

## Коротко
Автопереключение вкладок с гибкими таймерами, профилями, отдельным окном мониторинга и автообновлением выбранных страниц.

## Полное описание (пример)
- Настраиваемый интервал переключения + индивидуальные таймеры для каждой страницы.
- Профили (Work/Monitoring/Reports) с быстрым переключением и импортом/экспортом JSON.
- Отдельное окно мониторинга, чтобы не трогать рабочие вкладки.
- Автообновление выбранных страниц, исключения по доменам, случайный или последовательный порядок.
- Бейдж на иконке показывает обратный отсчёт (можно отключить).
- Ничего не отправляет наружу, настройки хранятся локально (см. PRIVACY_POLICY.md).

## Разрешения из манифеста

- `storage` — сохранение настроек и профилей локально.
