# Переменные окружения и секреты

В текущей версии расширения `tab-rotator` **не требуется** `.env` и нет серверных ключей.

## Правило
Даже если репозиторий закрытый, **не коммитьте прод‑секреты** (токены/пароли/ключи) в git.

Если в будущем появятся интеграции с внешними сервисами:
- храните реальные значения в локальном `.env` (он игнорируется через `.gitignore`),
- добавляйте только шаблон `.env.example` или список переменных без значений.
